# __Herc Grades__
## __Description__
An app that makes it easy for teachers to assign grades to their students!

### Ruby Version: 2.4.0

## __Dependencies__
* _Views_: React
* _Styling_: Bootstrap
* _Database_: Postgres
* _Testing_: RSpec
